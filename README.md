### ✉️ Commit Convention
> 학습 내용 정리 후 가능한 1커밋을 유지합시다!<br>
> 커밋 형식을 꼭 지킵시다!

아래 링크를 참고해서 message template 설정 후 commit을 작성합니다.  
해당 로컬 레포지토리 루트 위치에서 아래 명령어를 실행하여 템플릿을 적용할 수 있습니다. 
```bash
git config commit.template .gitmessage.txt
```
본문의 내용은 자유롭게 작성하되, 스터디 모임 중 리뷰 시 설명이 가능해야 합니다.  
[git commit message template 참고 링크](https://velog.io/@bky373/Git-%EC%BB%A4%EB%B0%8B-%EB%A9%94%EC%8B%9C%EC%A7%80-%ED%85%9C%ED%94%8C%EB%A6%BF)

커밋 메시지 제목은 아래 포맷을 권장합니다.
```text
[주차] 이름 : 어떤 내용인지 요약
EX) [1주차] 장규 : JAVA 기초 발표자료 업로드
EX) [3주차] 은성 : RealMySQL 05장 학습 내용 정리
```
